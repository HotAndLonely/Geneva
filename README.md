# HotL Overlay for Dead by Daylight

<h2>it counts the times a survivor is hooked</h2> 

<ul>
  <li>Press "1", "2", "3" or "4" to count</li>
  <li>Press "0" to reset</li>
  <li>Press "9" to exit</li>
  <li>Press "8" to enable mouse protection</li>
</ul>
